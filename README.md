# 3D-printer-air-filter
Scripts for arduino uno for modulating up to 3 fan speeds based on temperature feedback. 

## Hardware:

Arduion Uno rev3

Fans: 9S0912P4F011 Sanyo Denki

Temp Sensors: DS18B20
